BrainSweeper

  • About: A twist on MindSweeper. This program implements memory management, pointer manipulation, bitwise operations, and multi-dimensional arrays. Included functions for creating and cleaning a game board, computing the number of neighboring mines, and printing the board to the console. Utilized features such as function overloading, enumerations, and size_t datatype and other c++ standard libraries like iostream for input/output operations and 'cout' for printing.





    • Creed's Cookie Confectionery

  • About: A prototype store membership and selling program. This program implements memory management, pointer manipulation, polymorphism, exceptions, object-oriented programming, string manipulation, and multi-dimensional arrays. The program allows cashiers to make orders, create memberships, and edit memberships. Additionally, it informs the cashier and member of their point system with a printed receipt.

  • About: Implemented a flight booking system utilizing object-oriented programming, arrays for data management, and input validation.     • Tower of Hanoi

  • About: Implemented a simulation of the classic game 'Tower of Hanoi' in C++, utilizing features such as exception handling, string manipulation, and multi-dimensional arrays. Wrote custom exception classes to handle input validation, and implemented a user interface using standard input/output operations. Developed a game loop that allows the player to make moves and updates the game state based on user input, and provided an overview of the game state at each turn.
