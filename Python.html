Bomb Shell Game

  • About: This is a program that creates a pygame-based video game where a player controls a shield and moves it to catch bombs that are falling from the sky. The program uses pygame library to handle graphics, sound and event handling. There are several game functions and mechanics implemented such as collision detection, score keeping, and difficulty progression. The game also has the ability to keep track of the best score.
