Project Overview

5-Stage RISC-V Architecture

This project implements a 32-bit scalar processor based on the RV32I ISA. The design features a classic 5-stage pipeline (IF, ID, EX, MEM, WB) with full hazard detection and data forwarding logic to minimize stalls.
