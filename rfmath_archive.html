RFMath Library: Computational Electromagnetics

RFMath is a high-performance Python library designed for the rapid analysis of cascaded microwave networks. By leveraging NumPy-vectorized operations, the engine computes complex S-parameter responses across wide frequency sweeps without the overhead of traditional EDA GUI environments.

Core Engine Functionality
  • Vectorized Modeling: Functions like abcd_TL() generate frequency-dependent transfer matrices for distributed transmission lines, accounting for phase constant ($\beta$) and characteristic impedance ($Z_0$) over $N$ frequency points.
  • Network Synthesis: Implements abcd_Y_inverter() and abcd_series_Z() to model lumped elements and impedance transformers, essential for building complex filters and matching networks.
  • ABCD Cascading: The cascade() function utilizes optimized matrix multiplication to chain an arbitrary number of two-port networks into a single system matrix, effectively solving the entire circuit's behavior.
  • Parameter Transformation: Includes robust abcd_to_s() conversion logic, allowing the engine to translate raw ABCD parameters into measurable Scattering Parameters (S11, S21) relative to a system reference impedance.
