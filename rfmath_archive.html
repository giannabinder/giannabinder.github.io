Circuit Theory: The ABCD Model

This solver models a Lossless Transmission Line. Unlike low-frequency circuits, RF signals behave as waves. We use the ABCD Matrix to relate input ($V_1, I_1$) to output ($V_2, I_2$).

The matrix is defined by characteristic impedance ($Z_0$) and electrical length ($\theta = \beta \ell$). Multiplying matrices allows for seamless component cascading.
